In the original example the scriptblock was being bound to the callers scope which can not access anything in the module except what was imported into the scope. Get module Info for a Function in Powershell v2.0

Sun, Jan 3, 2010 

This one liner will return the the module info in a easy to consume format. The info includes the full path to the module that the function is defined in. Good Times and Get-Command!!!


Get-Command -CommandType Function -Name Control-FPService | Select-Object -ExpandProperty Module | Format-List
